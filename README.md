React Boilerplate made with Babel & Webpack 4

## Features
* Axios
* React-router
* Scss & Sass
* Import images
* js & jsx file format

## Commands

**To install**
------
```
git clone https://github.com/GabrielvonPlaten/React-Boilerplate.git
cd React-Boilerplate
npm / yarn install
```
serve
------
```
npm / yarn run serve
```


Local server will start at **http://localhost:8080**

build
------
```
npm / yarn run build
```

dev-server
------
```
npm / yarn run dev-server
```
